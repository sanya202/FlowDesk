# FlowDesk 🚀

FlowDesk is a full-stack project management application inspired by Jira. It helps teams organize work using workspaces, projects, and tasks.

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- bcrypt
- JWT (Coming Soon)

### Frontend (Coming Soon)
- React.js
- Tailwind CSS
- Axios

## Features

### Authentication
- ✅ User Registration
- ⏳ User Login
- ⏳ JWT Authentication

### Project Management
- ⏳ Workspaces
- ⏳ Projects
- ⏳ Tasks
- ⏳ Comments
- ⏳ Activity Logs
